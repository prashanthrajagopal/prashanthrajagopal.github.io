Setting Up ELK Cluster

Oct 31, 2015

We will be setting LogStash with all its required dependancies in this post. You must not face any issues hopefully. Do let me know if you do. It is pretty simple. It does have a lot of moving parts mentioned below, but it must be fine as long as you dont mess around with things you cannot handle.

Java – Logstash uses to run. Logstash – Collects and processes the logs. ElasticSearch – Stores, indexes and allows for searching of Logs. Redis – This is used as a queue and broker to feed messages and logs to logstash. Kibana – Web interface for searching and analyzing logs stored by ES.

Prerequisite

ElasticSearch

ELasticsearch can be installed from Yum and APT repositories.

Logstash

You can download the lastest logstash from https://www.elastic.co/downloads/logstash
